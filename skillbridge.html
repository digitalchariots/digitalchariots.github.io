SkillSwipe Hackathon Project

Swipe Through Skills
Not Resumes

A skills-first hiring concept built for the Creator Colluseum hackathon. The main idea is to match candinates with employers based upon skills, not CVs full of buzzwords.

Try the Demo

About This Project

This prototype was built to explore what skill-based hiring could look like, if the origin resume-based hiring system was phased out. The demo is a rough proof of concept with limited functionality.
